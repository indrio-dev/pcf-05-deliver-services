## University of Missouri Studies (PRO-Albrecht Evidence)

**Source:** [Acres USA - Proving Albrecht](https://members.acresusa.com/proving-albrecht/)
**Location:** Sanborn Field, University of Missouri (Albrecht's home institution)
**Researcher:** Reinbott

### POSITIVE FINDINGS (Albrecht WORKS):

**1. Yield Response to Balanced Ca/Mg:**
```
Balanced soil (68% Ca, 12% Mg):
  → Outperformed unbalanced treatments
  → Outperformed Mg-excess treatments

Corn and soybean yields: Responded positively
Greatest response: During DRY YEARS (stress conditions!)
```

**2. QUALITY Improvement (Not Just Yield):**
```
Corn crude protein:
  - Balanced soil: 8.1%
  - Control plots: 7.5%

Difference: +0.6% protein (8% increase in quality!)

This is QUALITY improvement, not just quantity!
```

**3. Soil Biological Response:**
```
Balanced soils showed:
  - Greater soil microbial abundance
  - Improved aggregate stability

With manure: Significantly enhanced microbial activity

Biology responds to balanced minerals!
```

**4. System Effects:**
```
Rotation (wheat included): Better performance
vs Continuous corn: Less response

Manure application: Enhanced all effects

Complete system approach works!
```

## Why Missouri (✅) vs Ohio State (❌) Got Different Results

### University of Missouri (POSITIVE):
```
Conditions:
- Balanced Ca/Mg (68:12 = 5.7:1, close to Albrecht 6.5:1)
- Rotation (corn, soy, wheat)
- Manure application (biology + OM)
- Measured quality (protein content, not just yield)
- Long-term plots (Sanborn Field = decades)

Result: POSITIVE response
  - Yield increase
  - Protein +0.6% (quality!)
  - Microbial abundance higher
  - Aggregate stability better
```

### Ohio State (NEGATIVE):
```
Conditions:
- Ca:Mg manipulation (gypsum only)
- High tillage annually (destroys aggregation)
- No biology amendments
- Conventional/organic NPK (not holistic)
- 4 years only (short term?)
- Already above sufficiency

Result: NO response
  - No yield effect
  - No soil health effect
  - Minimal physical effects
```

**The difference:**
```
Missouri: Complete system (minerals + biology + rotation + manure)
Ohio: Minerals in isolation (no biology, high tillage)

Missouri validated: Albrecht works when COMPLETE
Ohio validated: Albrecht fails when PARTIAL
```

## The Pattern Emerges (Evidence-Based)

### Albrecht WORKS When (✅):

**1. Active Biology Present:**
```
Missouri: Manure application (biology + OM)
Nutri-Tech: Humates, inocula, compost tea
BFA regenerative: Compost, biology-focused practices

Biology is REQUIRED to make minerals available!
```

**2. Soil Structure Preserved:**
```
Missouri: Rotation, reduced disturbance (long-term plots)
Nutri-Tech: Reduced tillage recommended
BFA regenerative: Often no-till or reduced

Structure allows Ca aggregation effects
```

**3. Addressing Deficiencies:**
```
Missouri: Building from baseline
Nutri-Tech: Fixing imbalances
BFA: Wide range of soils (some deficient)

Benefit when fixing problems, not adding to adequate
```

**4. Stress Conditions:**
```
Missouri: "Greatest responses during DRY YEARS"

When water/stress limits growth, minerals become limiting factor
In good years, not limiting
```

**5. Quality Measured (Not Just Yield):**
```
Missouri: Protein +0.6% (quality improvement!)
BFA: Brix +1.63°Bx (quality improvement!)

May affect quality MORE than quantity
```

---

### Albrecht DOESN'T WORK When (❌):

**1. No Active Biology:**
```
OSU: High tillage, no biology amendments
Result: Minerals can't be utilized without microbes

Minerals alone: Insufficient
```

**2. Structure Destroyed:**
```
OSU: Annual tillage destroys aggregation
Even if Ca promotes flocculation, tillage breaks it

Can't build what you destroy annually
```

**3. Already Above Sufficiency:**
```
OSU: Soils adequate in Ca/Mg
Adding more to adequate: No benefit

Law of diminishing returns
```

**4. Single Factor Tested:**
```
OSU: Changed Ca:Mg ratio only
Not holistic: Other factors not optimized

Albrecht is system, not single adjustment
```

**5. Short Timeframe:**
```
OSU: 4 years
Missouri: Decades (Sanborn Field long-term)

Structure/biology changes may take time
```

**Albrecht Method validated WHEN:**
```
✅ Complete system (minerals + biology + structure)
✅ Rotation and diversity
✅ Organic matter/manure included
✅ Reduced tillage (allows aggregation)
✅ Measuring quality (protein, Brix) not just yield
```

**Albrecht Method fails WHEN:**
```
❌ Minerals alone (no biology)

❌ Short timeframe (4 years may be insufficient)
❌ Already adequate soils (adding to sufficiency)
❌ Single factor manipulation (not holistic)
```
