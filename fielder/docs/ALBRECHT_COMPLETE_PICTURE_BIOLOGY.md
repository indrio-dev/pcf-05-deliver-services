
```

```

**3. Soil Structure (S Pillar - Physical):**
```
Aggregation (Ca flocculation)
Pore space (gas/water exchange)
Requires: Reduced disturbance (low/no till)

Destroyed by: Annual tillage (OSU issue!)
```

**ALL THREE together = Albrecht system works**

**Missing ANY = System fails**
